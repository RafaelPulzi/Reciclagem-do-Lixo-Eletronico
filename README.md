# ♻️ Reciclagem do Lixo Eletrônico

[![Licença](https://img.shields.io/badge/licença-MIT-blue.svg)](LICENSE)

## 📌 Sobre o Projeto

Este site foi desenvolvido com o objetivo de conscientizar sobre o descarte correto de lixo eletrônico e seus impactos negativos ao meio ambiente. O projeto aborda a importância da reciclagem de eletrônicos, os riscos do descarte inadequado e oferece orientações sobre como destinar esses resíduos de forma responsável.

## 🚀 Tecnologias Utilizadas

- **HTML5** – Estrutura do site
- **CSS3** – Estilização e layout responsivo
